# \# Oracle Memory Calculator (React + TypeScript)



##### \## 📌 Overview



###### This project is a simple Oracle Memory Calculator built using React and TypeScript. It helps estimate memory components such as SGA and PGA based on user inputs.



\---



##### \## ⚙️ Prerequisites



###### Make sure the following are installed on your system:

###### 

###### \* Node.js (v16 or above recommended)

###### \* npm (comes with Node.js)

###### 

###### To verify:

###### node -v

###### npm -v

###### 

###### \---



##### \## 📥 Installation Steps



###### \### 1. Clone the Repository

###### 

###### git clone https://github.com/sanjaikumar1/Oracle-Memory-Calculator.git

###### cd oracle-app

###### 

###### \---



##### \### 2. Install Dependencies



###### npm install

###### 

###### \---



##### \### 3. Install Additional Package (Required)



###### This project uses lucide-react for icons:

###### 

###### npm install lucide-react

###### 

###### \---



##### \### 4. Start the Application



###### npm start

###### 

###### \---



##### \## 🌐 Access the Application



###### Open your browser and go to:

###### http://localhost:3000



###### \---



##### \## 📁 Project Structure



###### src/

###### ├── App.tsx

###### ├── oracle\_memory\_calculator.tsx

###### ├── index.tsx



\---



##### \## ⚠️ Important Notes



###### \* Do NOT run .tsx files directly using tsx or node.

###### \* This is a React component and must run inside a React application.

###### \* If you see errors like:

###### &#x20; "Cannot find module 'lucide-react'"

###### &#x20; → Run: npm install lucide-react

###### 

###### \---



##### \## 🛠 Troubleshooting



###### \### Port already in use:

###### 

###### Kill the process or run on a different port.

###### 

###### \### App not loading:

###### 

###### Ensure npm start is running.
